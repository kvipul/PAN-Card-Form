App is deployed on Heroku: https://pan-card-form.herokuapp.com/ 
About:
	This Psuedo Web App is for filling the PAN card data like PAN no., Company Name and Date. We can fill the required data manually or by uploading file. PAN field accepts only alpha-numeric character and Date should be in YYYY-MM-DD format. Format of the file is also given on the web page.
